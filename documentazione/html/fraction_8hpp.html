Detailed Description

Definizione della classe Fraction per l'aritmetica razionale esatta.

La classe rappresenta un numero razionale p/q con numeratore e denominatore di tipo long long. La frazione è mantenuta sempre in forma ridotta ai minimi termini e con denominatore positivo, grazie alla normalizzazione eseguita nel costruttore tramite il massimo comun divisore (MCD).

Tutta l'implementazione è inline nel file .hpp; il file .cpp è vuoto per convenzione (classi piccole senza dipendenze esterne).

Note
Questa classe è usata da Capacity (capacity.hpp) per rappresentare capacità frazionarie esatte nei grafi di flusso DIMACS.
