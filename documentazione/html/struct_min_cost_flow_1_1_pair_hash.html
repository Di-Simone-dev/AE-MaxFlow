Detailed Description

Hash per std::pair<int,int> basato su codifica a 64 bit.

Combina i due interi in un int64_t via shift di 32 bit, garantendo assenza di collisioni per indici di nodo entro 2^31. Sostituisce la std::map originale (O(log n)) con lookup O(1) ammortizzato.
