Detailed Description

Implementazione dell'algoritmo Capacity Scaling (Gabow, 1985).

Calcola il flusso massimo su grafi orientati con capacità long long (modalità intera) o double (modalità virgola mobile). La modalità si sceglie tramite il costruttore appropriato.

Uso tipico (modalità intera)

CapacityScaling::IntGraph g;
g[{0, 1}] = 10;
g[{0, 2}] = 5;
g[{1, 3}] = 8;
g[{2, 3}] = 7;
CapacityScaling cs(g);
auto result = cs.max_flow(0, 3);
long long flow = std::get<long long>(result); // 13

CapacityScaling::CapacityScaling ( const IntGraph &  graph)
explicit

Costruisce un oggetto CapacityScaling in modalità intera.

Per ogni arco (u → v, cap) vengono aggiunti al grafo residuo:

  • un arco diretto con capacità cap;
  • un arco inverso con capacità 0.

Tutti i nodi sono registrati automaticamente.

Parameters
graphMappa degli archi con capacità intere.
Postcondition
max_flow() restituirà un std::variant con long long attivo.

Il costruttore esegue tre passi:

  1. Raccolta dei nodi: scorre tutti gli archi di graph e registra ogni estremo in _index (se non già presente), assegnandogli un indice interno progressivo 0-based.
  2. Allocazione della lista di adiacenza: ridimensiona _adj_int a _n liste vuote.
  3. Inserimento degli archi residui: per ogni arco (u → v, cap):
    • aggiunge a _adj_int[u] l'arco diretto con capacità cap;
    • aggiunge a _adj_int[v] l'arco inverso con capacità 0. I campi rev si puntano a vicenda per aggiornamenti O(1) in augment.

Costruisce un oggetto CapacityScaling in modalità double.

Identico al costruttore intero ma con capacità double. I confronti con zero usano la soglia CapacityScaling::EPS.

Parameters
graphMappa degli archi con capacità floating-point.
Postcondition
max_flow() restituirà un std::variant con double attivo.

Identico al costruttore per IntGraph, ma opera su _adj_dbl e imposta la capacità degli archi inversi a 0.0. Il flag _is_double viene impostato a true per indirizzare le chiamate successive a _max_flow_dbl.

int  sink 
)

Calcola il flusso massimo da source a sink.

Esegue l'algoritmo Capacity Scaling sul grafo residuo costruito dal costruttore. Il grafo residuo viene modificato in-place.

Il tipo del valore restituito dipende dal costruttore usato:

Costruttore Tipo nel variant Come accedere
IntGraph long long std::get<long long>(res)
DblGraph double std::get<double>(res)
Parameters
sourceIdentificatore del nodo sorgente.
sinkIdentificatore del nodo pozzo.
Returns
Flusso massimo come std::variant<long long, double>.
Exceptions
std::out_of_rangeSe source o sink non sono nel grafo.
Note
Se source == sink, restituisce 0 senza eseguire l'algoritmo.

Il metodo:

  1. Verifica che source e sink esistano in _index, lanciando std::out_of_range in caso contrario.
  2. Gestisce il caso degenere source == sink restituendo 0 senza eseguire l'algoritmo.
  3. Delega a _max_flow_int() o _max_flow_dbl() in base al flag _is_double impostato dal costruttore.
