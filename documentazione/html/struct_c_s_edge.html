Detailed Description

template<typename Cap>
struct CSEdge< Cap >

Arco nel grafo residuo di CapacityScaling, templatizzato sul tipo di capacità.

Struttura analoga a ResidualEdge di PushRelabel: ogni arco originale (u → v, cap) genera un arco diretto e uno inverso collegati tramite rev.

Template Parameters
CapTipo numerico della capacità: long long o double.
