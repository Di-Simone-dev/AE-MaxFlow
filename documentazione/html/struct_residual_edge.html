Detailed Description

template<typename Cap>
struct ResidualEdge< Cap >

Arco nel grafo residuo, templatizzato sul tipo di capacità.

Ogni arco originale (u → v, capacità c) genera due istanze di ResidualEdge:

  • arco diretto: {to=v, rev=idx_inv, cap=c}
  • arco inverso: {to=u, rev=idx_dir, cap=0}

Il campo rev permette di aggiornare la capacità dell'arco opposto in tempo O(1) durante le operazioni di push.

Template Parameters
CapTipo numerico della capacità: long long o double.
